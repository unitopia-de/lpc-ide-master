# Sample Workspace

Test-Workspace für die LPC-IDE-Master-Extension.

## Verwendung

1. F5 in der Extension drücken (Extension Development Host startet).
2. Im neuen Fenster: **File → Open Folder…** → diesen Ordner öffnen.
3. Die Extension sollte:
   - Auto-Discovery: `secure/master.c` finden → Dialekt `ldmud-unitopia`.
   - Statusleiste unten rechts: `LPC: ldmud-unitopia (N)`.
   - In der Activity Bar: **Remote MUD** Container.
4. `lpc-config.json` editieren — Änderungen werden automatisch neu geladen.
